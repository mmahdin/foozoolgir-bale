# Bale Bot Manager - Backend

## ویژگی‌های جدید (نسخه ۲.۰)

### ۱. دستور `/getlink` - لینک اختصاصی شخصی
کاربر با ارسال `/getlink` به ربات، یک لینک **منحصربه‌فرد** دریافت می‌کند که بر اساس UUID ساخته شده و قابل حدس نیست. این لینک را می‌تواند در بایو خود قرار دهد.

**جریان کار:**
1. مهدی دستور `/getlink` می‌فرستد
2. ربات لینک `https://ble.ir/BOT?start=abc123xyz` را به مهدی می‌دهد (توکن UUID)
3. مهدی لینک را در بایو خود می‌گذارد
4. کوثر روی لینک کلیک می‌کند
5. ربات به کوثر: «به مهدی می‌گم پروفایلش رو چک کردی»
6. ربات به مهدی: «کوثر پروفایل شما را دید!»

### ۲. ارسال/حذف پیام از پنل
- از بخش **کاربران** پنل می‌توانید به هر کاربر پیام بفرستید
- پیام‌های ارسالی در پنل نمایش داده می‌شوند
- می‌توانید پیام را حذف کنید (از چت کاربر هم پاک می‌شود)

### ۳. دسته‌بندی کاربران بر اساس لینک
- در بخش **لینک‌ها** می‌توانید ببینید چه کسانی از هر لینک آمده‌اند
- لینک‌های شخصی (/getlink) در تب جداگانه نمایش داده می‌شوند

### ۴. پیام‌های ویژه برای کاربران خاص
- در بخش **پیام‌های ویژه** می‌توانید برای یک کاربر خاص (مثلاً mahdi) متن سفارشی تعیین کنید
- شناسه کاربر می‌تواند **آیدی عددی** یا **یوزرنیم** باشد

## نصب و راه‌اندازی

```bash
cd backend
pip install -r requirements.txt
```

فایل `.env` را بسازید:
```env
BOT_TOKEN=توکن_ربات_شما
BOT_USERNAME=یوزرنیم_ربات
PORT=8000
```

اجرا:
```bash
uvicorn main:app --reload --port 8000
```

## API Endpoints جدید

| Method | Endpoint | توضیح |
|--------|----------|-------|
| GET | `/api/personal-links` | لیست لینک‌های شخصی |
| GET | `/api/personal-links/{token}/visitors` | بازدیدکنندگان یک لینک شخصی |
| POST | `/api/users/{id}/send-message` | ارسال پیام به کاربر |
| GET | `/api/users/{id}/sent-messages` | پیام‌های ارسالی به کاربر |
| DELETE | `/api/sent-messages/{id}` | حذف پیام ارسالی |
| GET | `/api/special-messages` | پیام‌های ویژه کاربران |
| POST | `/api/special-messages` | تنظیم پیام ویژه |
| DELETE | `/api/special-messages/{id}/{key}` | حذف پیام ویژه |

## ساختار فایل‌های جدید

```
backend/data/
├── bot_messages.json      # متون ربات (قابل ویرایش از پنل)

├── sent_messages.json     # پیام‌های ارسالی از پنل
├── special_messages.json  # پیام‌های ویژه کاربران خاص
├── links.json             # لینک‌های عمومی
├── users/                 # اطلاعات کاربران
├── messages/              # پیام‌های دریافتی
└── profile_photos/        # عکس‌های پروفایل
```
